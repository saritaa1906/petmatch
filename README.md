# 🐾 PetMatch — Conectando Corações, Mudando Vidas

O **PetMatch** é um sistema web administrativo moderno e responsivo desenvolvido para a gestão interna de ONGs de proteção animal e abrigos de pets. O sistema centraliza o controle de voluntários, o cadastro detalhado de animais para adoção e o mapeamento dinâmico de espécies e raças.

---

## ✨ Funcionalidades Principais

* 🔐 **Autenticação Segura:** Proteção de rotas administrativas com controle de sessão em PHP.
* 🐾 **Gestão de Pets (CRUD):** Cadastro, edição, listagem e exclusão de animais de forma simplificada.
* ⚡ **Filtros Dinâmicos:** Seleção inteligente de raças que se atualiza automaticamente de acordo com a espécie escolhida via JavaScript.
* 🎨 **Interface Premium:** Design fluido em *Dark Mode* com a identidade visual laranja do projeto.

---

## 🛠️ Tecnologias Utilizadas

* **Backend:** PHP 8.x
* **Banco de Dados:** MySQL / MariaDB (via `mysqli`)
* **Frontend:** Bootstrap 5.3, HTML5, CSS3 Customizado
* **Interatividade:** JavaScript Vanilla (Manipulação de DOM)
* **Versionamento:** Git & GitHub

---

## 🔧 Como Rodar o Projeto Localmente

1. Clone o repositório na pasta do seu servidor local (ex: `htdocs` do XAMPP).
2. Crie um banco de dados chamado `petmatch` no MySQL e importe as tabelas.
3. Configure as credenciais de acesso em `config/conexao.php`.
4. Abra o navegador e acesse: `http://localhost/petmatch/login.php`
